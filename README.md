# TCAdmin V3 - Issue Tracker

This repository is the **public issue tracker** for [TCAdmin V3](https://www.tcadmin.com). It is used to track bug reports, feature requests, and general feedback from the community.

**Source code is maintained in a separate private repository.**

## How to Use

- [**Browse existing issues**](https://github.com/TotalControlAdmin/TCAdminV3-Issues/issues)
- [**Report a bug**](https://github.com/TotalControlAdmin/TCAdminV3-Issues/issues/new?template=bug_report.yml)
- [**Request a feature**](https://github.com/TotalControlAdmin/TCAdminV3-Issues/issues/new?template=feature_request.yml)

## Guidelines

- Search existing issues before creating a new one to avoid duplicates.
- Provide as much detail as possible when reporting bugs (steps to reproduce, expected vs actual behavior, environment info).
- Be respectful and constructive in all discussions.
